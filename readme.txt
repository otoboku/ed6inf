空之轨迹三部曲 显敌我 战斗数值、抗性等 补丁

支持的游戏版本：
三豪华 简体中文版
三豪华 日文版：英雄徽h 空のEセット Windows 7 臧
3rd仿p系列：仿p、new3rd时之轨迹、The Other Side
3rd日文旧版1.002
整合补丁（理之补丁 自带显数值、抗性）
sc日文旧版1.020

功能：
1. 显敌我 能力值，第1列为裸值，第2列为总值，敌方的应该都一样
2. 显抗性，算上饰品的
3. 显CP
4. 显掉落物品的概率（FC SC）
5. 显掉落物品及概率、敌方战技名称（3rd）
6. 显AT及状态剩余AT
7. 仿p系列、整合补丁、理之补丁 AT显示减少为一列

可选功能：
1. 更改 战斗获得耀晶片上限(整合、理补除外)
2. 改变 敌方能力值


用法：
把dinput8.dll放到游戏目录，带上情报类回路看。
3rd掉落物品及概率、敌方战技名称 按PageUp/PageDown翻页键查看和切换。
可选功能用法见configEx.ini。


其它说明:
1. 状态变化并不会改变显示数值，(伤害、AT、命中)计算时才会加成


无效的情况下：
1. 请先确保是支持的版本
2. 假定"abcd.dll"无效，新建文本文件改名为"abcd.dll"（注意扩展名确实改掉），看游戏能否启动，应该会正常启动，否则请再次确保1。
（改注册表需谨慎，不懂不要弄，否则后果自负）
删除注册表"HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\KnownDLLs"下对应abcd的项
或者
在注册表"HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager"下新建“多字符串值”，
名字为"ExcludeFromKnownDlls"(不含引号，下同)，内容新增"abcd.dll"

然后 重启电脑即可。
